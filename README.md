# Studend board

System for the education.

# Clone

    git clone https://github.com/zdimon/student-board.git

# Installation

Rename sb/sb/_local.py to sb/sb/local.py 

    ./bin/install

Edit 

Create a simlink in sb/static directory.

    ln -s /home/webmaster/it-course/ course

# Run server

    ./bin/run
